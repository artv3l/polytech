# Сервис для анализа аудио-файлов

## Запуск приложения

```
docker compose up
```

## Открытые порты

- `http://localhost:8501` - Веб интерфейс пользователя
- `http://localhost:9090` - Мониторинг Prometheus
- `http://localhost:3000` - Мониторинг Grafana
- `http://localhost:8081` - Mongo Express
